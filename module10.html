Модуль контроля разовых посещений «QS-03»

Модуль билетного доступа по QR-кодам «QS-03» используется в турникетах для контроля доступа в кинотеатры, музеи, на выставкии, спортивные мероприятия и т.д. В качестве устройства идентификации здесь используется 2D сканер. Данное решение предназначено для разового прохода посетителей по приглашению, пропуску или билету, содержащему QR-код, а также для прохода с помощью смартфона с QR-кодом на экране.

Технические характеристики 2D сканера
  • Поддерживаемые стандарты: EAN-8, EAN-13, UPC-A, UPC-E, Code 39, Code 93, Code 128, EAN128, Codabar, Industrial 2 of 5, Interleave 2 of 5, Standard 25, Matrix 2 of 5, MSI, GS1, PDF417, MicroQR, DataMatrix, QR, HanXin, Aztec;
  • Скорость сканирования: 1300 раз в секунду;
  • Минимальный процент контрастности распознаваемого текста: 20%;
  • Допустимая освещенность окружающей поверхности: 0-120000 лк.
  • Напряжение питания: 5В;
  • Потребляемый ток: <200мА.

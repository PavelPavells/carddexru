Модуль гостевого доступа по QR-кодам «QRE-04» и «QRM-04»

Модули гостевого доступа по QR-кодам «QRE-04» и «QRM-04» предназначены для использования в турникетах, расположенных на различных объектах, где наряду с доступом авторизованных сотрудников, предусмотрен доступ сторонних посетителей.

В данном случае, гостевой доступ осуществляется посредством считывания QR-кодов со смартфонов посетителей или с бумажных носителей, распечатываемых и выдаваемых на проходной.

Данное решение является альтернативой устаревшей системы использования картоприемников для гостевого доступа и имеет ряд существенных плюсов, как по функционалу, так и по стоимости.

Технические характеристики 2D сканера
  • Поддерживаемые стандарты: EAN-8, EAN-13, UPC-A, UPC-E, Code 39, Code 93, Code 128, EAN128, Codabar, Industrial 2 of 5, Interleave 2 of 5, Standard 25, Matrix 2 of 5, MSI, GS1, PDF417, MicroQR, DataMatrix, QR, HanXin, Aztec;
  • Скорость сканирования: 1300 раз в секунду;

  • Допустимая освещенность окружающей поверхности: 0-120000 лк;
  • Напряжение питания: 5В;
  • Потребляемый ток: <200мА.

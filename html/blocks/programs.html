Бесплатное программное обеспечение CARDDEX IMS

Бесплатное программное обеспечение «Carddex IMS» разработано для контроля доступа, учета рабочего времени сотрудников и регистрации посетителей. Программа не требовательна к мощности ПК и может быть установлена почти на любом недорогом компьютере в локальной сети клиента.

«Carddex IMS» дает возможность пользователям работать без установки клиентской части программного обеспечения. Доступ к системе предоставляется через современный интернет-браузер с любого компьютера в локальной сети. Безопасность доступа обеспечивается авторизацией подключающегося клиента.

«Carddex IMS» поддерживает все модели выпускаемого оборудования Карддекс для систем контроля доступа, позволяет настраивать систему контроля, визуального управления оборудованием, онлайн-мониторинг событий системы.
